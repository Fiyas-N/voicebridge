# Environment Setup

## Quick Start

1. Copy `.env.example` to `.env`:
   ```bash
   copy .env.example .env
   ```

2. Fill in your actual API keys in `.env`

3. The `.env` file is git-ignored for security

## Required API Keys

- **Firebase**: Get from Firebase Console
- **Google Cloud**: For Speech-to-Text API
- **OpenAI**: For grammar analysis and feedback
- **Azure Speech**: For pronunciation assessment

See `quick_start_guide.md` for detailed setup instructions.
